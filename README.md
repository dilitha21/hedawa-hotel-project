# Hedawa Restaurant System

## Project Overview
The Hedawa Restaurant System is a responsive web application designed for a luxury eco-friendly traditional Sri Lankan hotel. This system provides a comprehensive digital experience enabling users to both view the restaurant's authentic food menu to place orders, and to browse premium hotel rooms (Single, Double, Suite) to make direct bookings. The platform combines a premium front-end aesthetic with robust backend management, featuring an integrated shopping cart that seamlessly handles pending food orders and room reservations together.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS3 (Custom Variables/Tokens), JavaScript.
- **Backend / Sever Logic**: Core PHP.
- **Database**: MySQL / MariaDB (via XAMPP).
- **Icons**: FontAwesome 6.
- **Typography**: Google Fonts (Poppins & Merriweather).

---

## Installation Guide
1. Copy the project folder into your XAMPP `htdocs` directory.
2. Start Apache and MySQL from the XAMPP Control Panel.
3. Import the `database.sql` file via phpMyAdmin.
4. Access the site at `http://localhost/hedawa-restaurant/`.
